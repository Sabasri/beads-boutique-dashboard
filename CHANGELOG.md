# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-28
### Added
- Complete Business Management Dashboard UI with React and Tailwind CSS.
- Express.js Backend with MongoDB integration and local JSON fallback.
- Secure JWT Authentication system.
- Dark/Light mode theme switching.
- Advanced Analytics page with Recharts.
- Real-time notification polling system.
- Product, Inventory, Customer, and Order management features.
- Premium jewelry-themed UI enhancements.
